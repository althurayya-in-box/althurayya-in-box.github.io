al-Ṯurayyā In-Box

Upload your data files to load the gazetteer and geospatial model.

The three required files are:

  • Places: a GeoJSON file with place features (places_new_structure.geojson)
  • Routes: a JSON/GeoJSON file with route sections (routes.json)
  • Regions: a JSON file with region definitions (regions.json)

al-Ṯurayyā In-Box

This is the in-box version of the al-Ṯurayyā project. Unlike the main project which loads its own data, this version allows you to upload your own data files and explore them with the same gazetteer and geospatial tools.

To load new data, click the button below.

Suggested citation: Seydi, Masoumeh, and Maxim Romanov. al-Ṯurayyā Project: a gazetteer and a geospatial model of the early Islamic World. (Based on: Georgette Cornu's Atlas du monde arabo-islamique à l'époque classique: IXe-Xe siècles, Leiden: Brill, 1983). [https://althurayya.github.io/], 2022—.
